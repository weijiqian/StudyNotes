### 大O表示法

**忽略常数,系数,低阶等.**

```
9 >> O(1)  
2n+3 >> O(n)
n2+2n+6 >>O(n2)
4n3 + 3n2 + 22n+100 >> O(n3)
写法上，n3 等价于 n^3 								
注意:大O表示法仅仅是一种粗略的分析模型，是一种估算，能帮助我们短时间内了解一个算法的执行效率 							 					 				
```

### fib函数时间复杂度分析

#### 1.O(2^n)

```java
public int fib1(int n){
  if(n<1) return n;
  return fib1(n-2) + fib1(n-1);
}
```

#### 2 O(n)

```java
public int fib2(int n){
   if (n < 1) return n;
  
  int first = 0;
  int second = 1;
  while(n-- > 1){
    second += first;
    first = second - first;
  }
  return second;
}
```
